# Projeto Boletim de Notas em Java :coffee:<br>

## O projeto visa trabalhar o funcionamento de um boletim escolar, onde nela o usuário poderá visualizar as descrições dos alunos, notas e a média.<br>

## Como fazer Rodar:<br>

1. Baixe o repositório remoto.
2. Abra no seu vscode ou em algum Editor de texto.
3. Rode o Programa.
4. Aparece no terminal as opções desejadas.
5. Escolha se deseja buscar um Aluno específico.
6. Digite o Método de busca, se é por nomes ou numero da matrícula.
7. O código mostrará o aluno em específico que você digitou senão aparecer irá aparecer a mensagem aluno nao encontrado.
